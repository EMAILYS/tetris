# 俄羅斯方塊實作完成！

我們已經成功在 `c:\tetris` 目錄下建立了一個純 HTML/CSS/JS 的俄羅斯方塊遊戲。

## 完成的檔案

- [index.html](file:///c:/tetris/index.html): 包含遊戲的整體結構與 Canvas 畫布。
- [style.css](file:///c:/tetris/style.css): 根據您的要求，我們實作了**暖色背景**（包含溫暖的米色/橘色按鈕等元素），並為方塊加入了發光效果 (Neon Glow)，讓介面更加現代化與精緻。
- [script.js](file:///c:/tetris/script.js): 遊戲的核心邏輯。

## 實作的功能重點

1. **七彩方塊**：7 種經典 Tetromino 分別對應 7 種鮮明的顏色。
2. **預覽面板**：右側面板可以顯示下一個即將出現的方塊。
3. **完整的操作控制**：
   - `←` / `→` : 左右移動
   - `↓` : 加速下落 (軟下落)
   - `↑` 或 `X` : 順時針旋轉 (右旋)
   - `Z` : 逆時針旋轉 (左旋)
   - `Space (空白鍵)` : **硬下落** (直接瞬間下落到底部)
4. **原生合成音效 (Web Audio API)**：不需要下載任何外部 mp3 檔案，完全由瀏覽器動態產生 8-bit 風格的音效！
   - 移動、旋轉會有清脆的短音。
   - 硬下落會有低沉的敲擊聲。
   - 消除行會有高音階的和弦效果。
   - 遊戲結束時會有漸弱的復古 Game Over 提示音。
5. **等級與計分**：隨著消除的行數增加，等級會提升，方塊下落的速度也會跟著變快。
